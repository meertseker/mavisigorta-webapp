# PWA Icons

Bu klasör PWA (Progressive Web App) için gerekli icon dosyalarını içermelidir.

## Gerekli İconlar:

- **icon-192.png** (192x192px) - Android için standart icon
- **icon-512.png** (512x512px) - Android için büyük icon
- **apple-touch-icon.png** (180x180px) - iOS için
- **favicon.ico** (32x32px) - Browser favicon

## Icon Oluşturma:

1. Ana logo dosyanızı hazırlayın (minimum 512x512px, tercihen SVG)
2. Online araçlar kullanarak gerekli boyutları oluşturun:
   - https://realfavicongenerator.net/
   - https://favicon.io/
   
3. Oluşturulan dosyaları bu klasöre yerleştirin

## Placeholder Kullanımı:

Geliştirme aşamasında, gradient placeholder icon'lar oluşturabilirsiniz:
- Tailwind CSS gradient renkleri kullanarak basit PNG dosyaları
- Canva veya Figma ile hızlı tasarımlar

**Not:** Production'a geçmeden önce profesyonel logo ile değiştirin!
